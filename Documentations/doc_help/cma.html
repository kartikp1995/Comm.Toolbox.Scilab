cma

This function creates an AdaptiveAlgorithm Object for a Constant Modulus Algortihm to use it with `lineareq` or `dfe` to create an equalizer object.

Calling Sequence

alg = cma(stepsize, leakage_factor)

Parameters

stepsize:

double-Any non-negative real number. It is StepSize parameter for CMA algorithm.

leakage_factor:

double-Real number between 0 to 1. It is Leakage Factor parameter for CMA algorithm. (Default = 1)

Examples

alg = cma(0.1,0.001)
