marcumq

This function calculates Marcum's Q function.

Calling Sequence

q = marcumq(a, b, m)

Parameters

a_vec:

Scalar or Vector or Matrix of values of a

b_vec:

Scalar or Vector or Matrix of values of b

m:

Scalar or Vector or Matrix of values of m

Examples

y = marcumq(1,2)
y = marcumq(1,2,2);
