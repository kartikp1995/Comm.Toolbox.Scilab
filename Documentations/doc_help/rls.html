rls

This function creates an AdaptiveAlgorithm Object for a Recursive Least Square Algorithm to use it with `lineareq` or `dfe` to create an equalizer object.

Calling Sequence

alg = rls(forgetfactor, invcorr0)

Parameters

forgetfactor:

Any non-negative real number. It is forgetfactor parameter for RLS algorithm.

invcorr0:

Real number between 0 to 1. It sets initialisation parameter for inverse correlation matrix. (Default = 1)

Examples

alg = rls(0.1)
